# Animal Tales Kids Support Page

Use this file set to publish a GitHub Pages support URL for App Store Connect.

## Fastest way

1. Create a new public GitHub repository such as `animal-tales-kids-support`.
2. Upload `index.html` from this folder to the root of that repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Set:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Save and wait for GitHub Pages to publish.

Your support URL will look like:

`https://sukheshcs.github.io/animal-tales-kids-support/`

## Important

- Replace `support@animaltaleskids.com` in `index.html` with your real support email.
- If you do not own a domain yet, you can temporarily use a Gmail address.
- App Store Connect support URLs must be public and load without logging in.
